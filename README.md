# QnA Now

NextJS14, tyepscript, supabase, genai, tailwind CSS, redux-toolkit, react-query
<br>
[사이트](https://qua-now.vercel.app/)
<br>
자유롭게 랜덤한 질문과 답을 받을 수 있는 사이트입니다.<br>
